集水域の作成

このツールは、解析レイヤの各ポイントを対象とした集水域または上流域を区別するエリアを決定します。

たとえば、水系汚染を表すポイント フィーチャがあり、可能性のある汚染源を特定する必要があるとします。 汚染源は、ポイントにおける集水域の上流域内のどこかにあるはずなので、このツールを使用して、汚染源を含む集水域を定義します。

解析ポイントの場所が重要です。 それらが排水管路に正確に位置していない場合、生成される集水域は、非常に小さくなり、解析に役立たない可能性があります。 [最近隣の排水管までの距離を検索] を使用すると、解析ポイントの場所を調整できるため、それらのポイントを排水管路に正確に一致させて、排水管路に位置するポイントに流れ込む集水域を特定することができます。 集水域を特定するために調整された場所は、出力レイヤに書き込まれます。入力解析ポイントは変更されません。

結果は、集水域と集水域の計算に使用される調整されたポイントの 2 つのレイヤを含むグループ レイヤになります。

生成される集水域には、入力ポイントのすべての属性の他に、次の属性も含まれます。

  • PourPtID - 入力ポイントからの ID 値が含まれます。
  • Area Square Kilometers - 上流集水域の面積（平方キロメートル）が含まれます。
  • Description および DataResolution - 経路の計算に使用される水文解析データに関連する情報が含まれます。

[現在のマップ表示範囲を使用] をオンにすると、現在のマップ範囲のフィーチャのみを使用して集水域を計算します。 オフの場合、解析レイヤのすべてのフィーチャが、現在のマップ範囲内になくても解析されます。

最近隣の排水管までの距離を検索

解析ポイントが排水管路から離れた場所にある場合、生成される集水域は非常に小さくなり、上流の汚染源を特定するのにあまり役立たない可能性があります。 排水管路上にあるポイントに流れ込む集水域を特定するには、ほとんどの場合において、解析ポイントを最近隣の排水管路にスナップすることが推奨されます。 最近隣の排水管路を特定するには、検索距離を指定します。 検索距離を指定しない場合、ツールは保守的な検索距離を計算して使用します。

入力ポイントの正確な場所を使用するには、検索距離をゼロに指定します。

解析目的の場合、排水管路は、標準的な水文解析モデルを使用して Esri によって事前に計算されています。 検索距離内に排水管路がない場合、最も大きな累積流量を含む検索距離内の場所が使用されます。

結果レイヤの名前

これは、 [マイ コンテンツ] で作成され、マップに追加されるレイヤの名前です。 デフォルト名は、解析タイプと解析レイヤ名に基づきます。レイヤがすでに存在する場合は、別の名前を指定するよう求められます。

[出力の保存先] ドロップダウン ボックスを使用して、結果を保存する [マイ コンテンツ] 内のフォルダの名前を指定できます。
