Вычисление плотности

Инструмент Вычисление плотности создает карту плотности из точечных или линейных пространственных объектов посредством разброса по карте известного количества событий (представленных как атрибуты точек или линий).Результатом является слой из полигонов, которые систематизированы от меньшей плотности к более высокой.

Каждая входная точка должна представлять местоположение некоторого события или инцидента, а полученный слой результатов будет представлять некоторое множество событий на единицу площади.Более высокое значение плотности в новом местоположении означает, что вблизи этого местоположения находится большее количество точек.Зачастую этот полученный слой может рассматриваться как зона (поверхность) повышенной вероятности для будущих событий.Например, если входные точки представляют местоположения ударов молний, то полученный слой может рассматриваться как зона (поверхность) повышенного риска ударов молний в будущем.

Поверхность плотности входных линий представляет общую величину линий, находящихся вблизи каждого местоположения.Единицей расчетного значения плотности является длина линии (линий) на единицу площади.Например, если линии представляют реки, то полученный слой будет представлять общую длину рек, не выходящих за пределы радиуса поиска.Такой результат может использоваться для определения областей, пригодных для обитания травоядных животных.

Другие примеры использования этого инструмента:

  • Создание карт плотности преступлений для помощи полиции в размещении ресурсов для снижения уровня преступности.
  • Вычисление плотности больниц в районе. Слой результата будет показывать области с высокой и низкой степенью доступности больниц, эта информация может быть использована для определения места под новую больницу.
  • Поиск областей с высоким риском лесных пожаров строится на основе исторических сведений.
  • Определение регионов, которые находятся далеко от крупных автомагистралей, может пригодиться при планировании строительства новых дорог .

Использование поля подсчета

Укажите поле, определяющее число событий в каждом местоположении.Например, если у вас города представлены точками, то в качестве поля подсчета вы можете использовать поле, представляющее население города; полученный слой плотности населения покажет более высокие плотности населения вблизи городов с большей численностью населения.

При использовании опции по умолчанию Не выбирать (None) предполагается, что каждое местоположение представляет одну единицу счета.

Расстояние поиска

Введите расстояние, в пределах которого будет выполняться поиск точечных или линейных объектов при вычислении значений плотности.Например, если вы укажете для поиска значение расстояния в 10000 метров, то в выходном слое при вычислении плотности любого местоположения будут учитываться объекты, находящиеся в пределах 10000 метров от этого местоположения.Если на расстоянии 10000 метров от местоположения событий нет, то такое местоположение получит нулевое значение плотности.

Если расстояние не задано, то по умолчанию при расчете плотности будут учитываться местоположения входных объектов и значения в поле подсчета (если таковое имеется).

Вырезать результат по

Нарисуйте на карте или укажите слой с областью, для которой необходимо рассчитать плотности.Например, при интерполяции плотностей популяции рыбы в озере можно указать контур озера, и тогда результат будет представлен в пределах границ этого озера.

Классифицировать по

Выберите, как классифицировать значения плотности.

  • Равные интервалы – Территория разделяется на области таким образом, чтобы диапазон значений в каждой из них был одинаковым.
  • Геометрический интервал – Области определяются на основании интервалов классов, которые имеют геометрические серии. Это позволяет добиться того, что в каждом диапазоне классов будет находиться примерно равное количество значений, и величина изменений по интервалам будет постоянной.
  • Естественные границы – Диапазоны определяются на основании естественной группировки данных. Значения границ классов определяются таким образом, чтобы сгруппировать схожие значения и максимально увеличить различия между классами.
  • Равная область – области создаются таким образом, чтобы размер каждой области был одинаковым. Например, если в полученном слое больше значений высокой плотности, это значит, что будет создано больше областей с высокой плотностью.
  • Стандартные отклонения – Области создаются на основании стандартных отклонений предсказанных значений плотности.

Число классов

Задает число классов (диапазон прогнозируемых значений) в слое результатов.Каждый класс определяет границы областей.Значения этих классов определяются параметром Распределить по (Classify by), упомянутым выше.

Выходные единицы площади

Укажите единицу площади.Плотность это – численность, поделенная на площадь, и этот параметр определяет единицу площади в расчете плотности.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от типа анализа и имени слоя анализа.Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
